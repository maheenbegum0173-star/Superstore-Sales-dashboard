# Superstore-Sales-dashboard

## Overview

This project is an interactive **Power BI Business Analytics Dashboard** designed to analyze sales performance, profitability, customer behavior, and product insights. The report provides multiple analytical views that help users understand business performance through KPIs, charts, and detailed analysis pages.

The dashboard enables decision-makers to track important metrics, identify trends, analyze customer segments, evaluate product performance, and make data-driven decisions.

---

## Project File

**File Name:** `Final__Project.pbix`

**Tool Used:** Microsoft Power BI Desktop

**Report Type:** Interactive Business Intelligence Dashboard

---

## Dashboard Pages

The report contains the following analytical sections:

### 1. Executive Overview

Provides a high-level summary of business performance.

Key insights include:
- Overall business KPIs
- Sales trends
- Performance summary
- Category-level comparisons
- Quick view of major business indicators

---

### 2. Sales Analytics

Focuses on understanding sales performance and trends.

Features:
- Sales analysis over time
- Sales distribution by categories
- Regional/category comparisons
- Interactive filters and slicers
- Sales trend visualization

---

### 3. Profit Analytics

Analyzes profitability and business efficiency.

Features:
- Profit performance tracking
- Profit comparison across categories
- Profit contribution analysis
- Profitability trends
- Category-wise performance insights

---

### 4. Customer Analytics

Provides insights into customer behavior and segmentation.

Features:
- Customer distribution analysis
- Customer-based performance metrics
- Customer segment comparison
- Buying behavior insights

---

### 5. Product Analytics

Analyzes product-level performance.

Features:
- Product sales analysis
- Product category comparison
- Product contribution trends
- Performance monitoring

---

### 6. Product Detail Analysis

Provides detailed product-level information.

Features:
- Product detail table
- Product-specific metrics
- Product performance comparison
- Drill-through analysis

---

### 7. Customer Detailed Analysis

Provides deeper customer-level insights.

Features:
- Customer details
- Customer performance analysis
- Customer comparison
- Detailed business metrics

---
